# سامانه مدیریت پروژه با Flask

دموی این سامانهٔ ساده مدیریت پروژه پیاده‌سازی‌شده با Flask.  
در این مخزن یک ویدیوی نمونه و مستندات خلاصهٔ پروژه قرار گرفته است.

## دمو
- ویدیوی دمو: (لینک ویدیو — مثلاً YouTube یا Release)
- پیشنهاد: اگر ویدیو را در YouTube قرار می‌دهی، حالت Unlisted را انتخاب کن و لینک را در اینجا بگذار.

## خلاصه
این برنامهٔ نمونه امکانات پایهٔ مدیریت پروژه را نمایش می‌دهد:
- مدیریت پروژه‌ها و کارفرماها
- برنامه‌ریزی فازها (گانت) با چک‌لیست‌های وزن‌دهی‌شده
- ثبت تراکنش‌های مالی با آپلود سند
- ثبت گزارش‌های پیشرفت فازها با امکان پیوست سند
- نمودارهای تعاملی پیشرفت برنامه‌ای و واقعی (Plotly)
- پروفایل کاربری با آپلود آواتار

## قابلیت‌های کلیدی
- نقش‌های کاربری: مدیر سیستم، مدیر پروژه، حسابدار، همکار
- تعریف پروژه، فاز و چک‌لیست با وزن
- گانت چارت با بررسی جمع وزن‌ها
- ثبت تراکنش با فایل پیوست
- ثبت گزارش پیشرفت با جلوگیری از کاهش درصد
- آرشیو اسناد پیوست شده
- پروفایل و تغییر رمز
- نمودارهای تعاملی و خط عمودی نمایش «امروز»

## فناوری‌ها
- بک‌اند: Python 3 + Flask
- ORM: SQLAlchemy (SQLite برای دمو)
- نمودار: Plotly
- فرانت‌اند: Bootstrap + CSS سفارشی

## ساختار پروژه (خلاصه)
- `app.py` — اپلیکیشن و روت‌ها
- `models.py` — مدل‌های دیتابیس
- `templates/` — قالب‌ها (HTML)
- `static/` — فایل‌های CSS/JS و تصاویر
- `uploads/` — فایل‌های آپلود شده
- `requirements.txt` — وابستگی‌ها

## نکات مهم
- برای محیط تولید از دیتابیس قدرتمندتر و مایگریشن استفاده کن.
- `SECRET_KEY` را در متغیر محیطی تنظیم کن.
- ممکن است بخواهی از سرویس ابری برای ذخیرهٔ اسناد استفاده کنی تا بار ذخیره‌سازی روی سرورت کم شود.
