فارسی — توضیحات کوتاه
- این ویدیو یک دموی کوتاه (حدود 3 دقیقه) از سامانهٔ مدیریت پروژه است که امکانات کلیدی را نشان می‌دهد:
  - تعریف پروژه و فازها (گانت)
  - ثبت تراکنش‌های مالی و آپلود اسناد
  - ثبت گزارش‌های پیشرفت با آپلود مدارک
  - مقایسهٔ پیشرفت برنامه‌ای و پیشرفت واقعی با نمودارهای تعاملی
- برای دیدن جزییات بیشتر به README اصلی مخزن مراجعه کنید.

English — Short description
- This short demo (approx. 3 minutes) showcases the Project Management System built with Flask:
  - Creating projects and phases (Gantt)
  - Logging financial transactions with document uploads
  - Submitting progress reports with attachments
  - Visualizing planned vs actual progress with interactive charts
- For more details and code structure, see the main README in the repository.
